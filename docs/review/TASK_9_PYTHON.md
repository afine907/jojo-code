# Agent 9: Python代码审查

## 审查范围

审查所有 Python 代码的 Pythonic 风格和最佳实践。

## 审查文件

```
src/nano_code/
├── server/
│   ├── rpc.py          # JSON-RPC Server
│   └── protocol.py     # 协议定义
├── tools/
│   ├── file_tools.py   # 文件工具
│   ├── shell_tools.py  # Shell工具
│   └── web_tools.py    # 网络工具
├── core/
│   ├── config.py       # 配置
│   ├── exceptions.py   # 异常
│   └── result.py       # Result类型
└── context/
    └── lazy_ignore.py  # 懒加载
```

## 审查要点

### 1. Pythonic 风格

检查项:
- [ ] 是否使用列表推导式
- [ ] 是否使用上下文管理器
- [ ] 是否使用生成器
- [ ] 是否使用装饰器

### 2. 类型注解

检查项:
- [ ] 函数参数类型
- [ ] 函数返回类型
- [ ] 类属性类型
- [ ] 是否使用 typing 模块

### 3. 资源管理

检查项:
- [ ] 文件是否使用 with
- [ ] 连接是否正确关闭
- [ ] 锁是否正确释放
- [ ] 异步资源是否正确清理

### 4. 并发安全

检查项:
- [ ] 是否有竞态条件
- [ ] 是否有死锁风险
- [ ] 是否正确使用锁
- [ ] 是否有线程安全问题

### 5. 代码组织

检查项:
- [ ] 模块职责是否单一
- [ ] 类是否过大
- [ ] 函数是否过长
- [ ] 是否有重复代码

### 6. 错误处理

检查项:
- [ ] 异常类型是否具体
- [ ] 是否有异常链
- [ ] 是否正确使用 finally
- [ ] 是否有资源泄漏
